# SSDP-Search
A light weight java respository that allows for the discovery of hardware devices on the local area network using the Simple Service Discovery Protocol (SSDP) as outlined in [UPnP™ Device Architecture 1.1](http://www.upnp.org/specs/arch/UPnP-arch-DeviceArchitecture-v1.1.pdf).

## Features:
* Create Formatted SSDP Search Requests
* Send SSDP Search Reuqest to Devices on the Network
* Parse any Recieved Responses
